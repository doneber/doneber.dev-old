## Quick start

1.  **Clone the repository.**

    ```shell
    git clone
    ```

2.  **Install dependencies.**

```shell
npm install
```

3.  **Start the development server.**

```shell
npm run dev
```

4.  **Open the source code and start editing!**

Your site is now running at `http://localhost:3000`!
